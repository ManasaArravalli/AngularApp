jQuery Animate

The jQuery animate() method is used to create custom animations. The required params parameter defines the CSS properties to be animated. The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds. The optional callback parameter is a function to be executed after the animation completes.

Syntax: $(selector).animate({params},speed,callback);
